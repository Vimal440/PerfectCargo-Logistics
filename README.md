# Perfect Cargo & Logistics

A modern, single-page application (SPA) built for Perfect Cargo & Logistics, a freight forwarding and custom clearance consultancy based in Rajkot, Gujarat, India. The website features a fully responsive design, custom scroll animations, and a dynamic user interface tailored for the logistics industry.

## 🌟 Features

* **Interactive Hero Carousel:** Engaging image slider with a continuous Ken Burns zoom effect and crossfade transitions.
* **Responsive Navigation:** A dynamic top navbar for desktop users and a mobile-friendly, app-style bottom navigation bar for smaller screens.
* **Scroll Reveal Animations:** Smooth, staggered entry animations that trigger as the user scrolls down the page.
* **Logistics Resources:** Dedicated sections outlining freight capabilities, mandatory shipping documents, and Incoterms® 2026.
* **Modern UI/UX:** Utilizes glassmorphism (frosted glass effects), gradient overlays, and clean typography (Oswald & Roboto).

## 🛠️ Technologies Used

This project was built focusing on simplicity and quick deployment without the need for a complex Node.js build pipeline:

* **Core:** HTML5
* **Framework:** React 18 (via CDN)
* **Styling:** Tailwind CSS (via CDN)
* **Icons & Fonts:** Font Awesome 6.4, Google Fonts
* **Compiler:** Babel Standalone (for in-browser JSX compilation)
